◆ ReadConfigFile()

void XPMP2::GlobVars::ReadConfigFile ( )

Read from a generic XPMP2.prf or XPMP2.<logAcronym>.prf config file.

Goal is to provide some few configuration options independend of the plugin using XPMP2. This is useful e.g. when configuring non-standard network settings for remote connections.
For even greate flexibility the function will try a plugin-specific file first, named "XPMP2.<logAcronym>.prf" (with spaces replaced with underscores) and if that is not found it tries the generic file "XPMP2.prf".
A config file is completely optional, none is required.
